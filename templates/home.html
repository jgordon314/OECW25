What is this?

This website provides suggestions based on what you already have or like.
Currently, we can give you food and movie suggestions.
To see food suggestions, select Fridge. To see movie suggestions, select Cinema.

Technical Description

Fridge takes in a list of ingredients and queries the Edamam API with the list of ingredients. Edamam provides recipes alongside the images and links to them, which are taken and displayed.

Cinema takes in a list of movies and finds the matching movie title using difflib. Using a pandas database containing a a pre-processed version of the GroupLens Research ml-latest dataset, a correlation of each input movie is found. These correlations contain how similar each input movie is to every single other movie in the database. These correlation tables are all taken and multiplied with each other to get the movies that are most similar to every input movie and is returned to the user.
